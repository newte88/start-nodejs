静态文件服务器

目的

  • 熟悉 nodejs.
  • 了解 web 相关知识.
  • 掌握 callback 处理过程.

实现

  • list 目录存储 markdown 静态文件.
  • 侦听 list 目录, 变化时自动更新 index 文件.
  • 使用 markdown 解析器, 生成 html 文件.

其它

  • 使用 Mou 编辑器中的 css 文件.
  • 使用 jQuery 中的 Callback, Deferred 模块处理回调.
