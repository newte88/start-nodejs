#nodeJS - 笔记#

**npm**: node package manager

####模块####
1. 文件系统 - fs
2. 模块 - module
3. 操作系统 API - process.
4. 网络通信 - http

CommonJS 规范

* 模块 - modules
* 包 - pakeages
* 系统 - system
* 二进制 - binary
* 控制台 - console
* 编码 - encodings
* 文件系统 - filesystems
* 套接字 - sockets
* 单元测试 - unit testing

JIT: Just-in-time Compilation

Node.js 最大的特点就是采用异步式 I/O 与事件驱动的架构设计。
